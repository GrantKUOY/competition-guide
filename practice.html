🌐 網路世界停看聽
每天滑手機，你是否曾懷疑過某則新聞是假的？假訊息（Fake News）已成為現代最嚴重的資訊危機之一。

請以「網路世界停看聽」為題，製作一份完整專題簡報，說明假訊息的成因與危害，並提出辨識方法與因應對策，讓更多人成為有判斷力的數位公民。
必要內容（對應 35% 資料結構分）
封面：主題標題 + 視覺圖 + 創用CC授權（左下角，5分直接送）
目錄：五個章節 + 超連結跳頁；每頁右上角「回目錄」按鈕
主題一：什麼是假訊息？（定義 + 3種常見類型）
主題二：假訊息如何傳播？（社群媒體演算法 + 同溫層效應）
主題三：我們能怎麼做？（查核工具 + 停看聽三步驟）
結語：1段行動呼籲，回應前言
⭐ 資料來源（必填！影響35%）——每筆資料一一列出
得分策略
版面 25%
每頁圖片≥60%，標題≥36pt，正文≤50字，不貼大段文字
手法 5%
用數字放大法（把關鍵數字放到80pt）或比較表格，不要純文字
配色 5%
全簡報最多2–3色，各章節用不同強調色，視覺有層次

🚇 大眾公共運輸

請製作一份完整專題簡報，說明大眾公共運輸的現況、優點、挑戰與未來發展，並提出鼓勵大家多搭大眾運輸的具體建議。
必要內容
封面 + 目錄（超連結）+ 前言
主題一：台灣大眾運輸種類與特色（捷運/公車/高鐵/公共單車）

結語 + 行動呼籲
⭐ 資料來源（必填！）
進階挑戰
挑戰①
主題二製作「各交通工具CO₂排放比較表格」
挑戰②
主題三做一頁數字強調頁：最震撼的數據放大到80pt
挑戰③

🌏 2050年台中永續生活場景
2050年，台中市已實現碳中和。風力渦輪、太陽能板、空中農園、綠色捷運……城市與自然和諧共存。

請畫出你想像中的2050年台中生活場景，可以是街道、公園、交通、學校或家庭角落。畫面中需有至少2個人物，呈現人與永續環境互動的日常。
必要元素
畫面尺寸：1280×720像素（橫式）
人物：至少2個，動作各不同（不要只站立）
前景／中景／背景：三層次清楚
永續元素：至少3種（太陽能、綠建築、單車、植物牆等）

加光暈或發光效果（sparkle），讓畫面更有層次感
配色 +分
暖色主角 × 冷色背景，互補色對比，視覺強烈
主題 +分
永續議題與當年SDGs熱門主題高度吻合，評審印象深

♻️ 垃圾分類大挑戰
台灣的垃圾分類做得如何？你能在時間壓力下，把垃圾正確分到對的桶子嗎？

請製作一個垃圾分類互動遊戲：畫面上方隨機出現垃圾物品（寶特瓶、廚餘、電池……），玩家在限時內拖曳到正確回收桶，答對得分、答錯扣分。
必備功能
開始畫面：遊戲標題 + 說明 + 開始按鈕
至少5種垃圾物品，隨機出現
至少3種回收桶（資源回收、廚餘、一般垃圾），標示清楚
計分：答對 +10分，答錯 -5分，即時顯示
倒數計時30秒，時間到顯示結果畫面
結果畫面：總分 + 重玩按鈕
⭐ 心智圖（必交）：角色清單、變數清單、流程圖
核心積木架構
🔄
廣播機制：「遊戲開始」「垃圾出現」「答對」「答錯」「時間到」
📦
清單：建立「垃圾種類」清單，隨機取第 X 項顯示
🖱️
拖曳判斷：放下時 → 碰到桶子 → 廣播對應結果
⏱️
計時器：重設計時器 → 等待直到計時器 > 30 → 廣播時間到
⏰ 3小時分配建議
📝
09:00–09:20｜畫心智圖（角色/變數/流程）
🎨
09:20–09:45｜製作角色：垃圾物品×5+、回收桶×3、背景
⚙️
09:45–10:30｜主流程：廣播架構、開始畫面、遊戲主迴圈
🎮
10:30–11:20｜完成計分、計時、判斷答對答錯
✨
11:20–11:40｜美化：音效、動畫、結果畫面
💾

🥗 健康飲食小測驗
你知道每天應該攝取多少蔬菜？哪種食物含有最多維他命C？

請製作一個健康飲食知識問答遊戲：出現題目與3個選項，玩家點擊作答，答對得分並給予鼓勵訊息，5題結束後顯示最終分數與評語。
必備功能
開始畫面 + 遊戲說明
至少5道題目，每題3個選項
答對顯示「答對！+10分」動畫，答錯顯示正確答案
題號進度顯示（第X題 / 共5題）
結果畫面：總分 + 3種評語（依分數不同）+ 重玩
⭐ 心智圖（必交）
核心積木架構
📦
用兩個清單：「題目」清單 + 「答案」清單，用題號對應
🔄
廣播「下一題」→ 從清單取出 題號 項 → 顯示
🖱️
當選項按鈕被點擊 → 比對清單第 題號 項 → 判斷對錯

🚦 路口交通安全模擬
每天上學要過馬路，你知道什麼時候該走、什麼時候要停嗎？

請製作一個交通安全模擬遊戲：玩家控制行人角色過馬路，紅燈時移動會被扣分，安全到達對岸得分，倒數30秒計算最終得分。
必備功能
行人角色：玩家用方向鍵或WASD控制移動
燈號系統：每3秒自動切換紅燈/綠燈，有視覺提示
違規判斷：紅燈時行人在馬路上 → 廣播違規 → 扣20分 + 回起點
倒數計時30秒，時間到顯示分數
汽車角色：綠燈時向左移動，增加難度
⭐ 心智圖（必交）
核心積木架構
🔄
三件事同時執行：燈號切換 / 倒數計時 / 玩家控制，各自獨立迴圈
📦
變數：「燈號狀態」（紅/綠）、「分數」、「計時器」、「過馬路次數」
⚠️
違規偵測：重複偵測「碰到馬路且燈號=紅」→ 立即廣播違規
